# 🐸 Rozitta Parser (v3.6 Beta)

**Rozitta Parser** — это десктопный инструмент на Python (PySide6 + Telethon), предназначенный для создания глубоких локальных архивов Telegram-данных. 

Проект создан как «цифровая страховка» и инструмент для подготовки данных к анализу в нейросетях, таких как **Google NotebookLM**, **ChatGPT** и **Claude**.

---

## ✨ Основные возможности

- 📁 **Полный бэкап:** Экспорт истории сообщений из личных чатов, групп и каналов (включая темы/topics на форумах).
- 🧠 **AI-Ready Export:** Специальный режим подготовки данных для ИИ. 
  - Экспорт в **Markdown (.md)** и **JSON**.
  - Автоматическое разбиение (chunking) файлов по ~300,000 слов для обхода лимитов NotebookLM.
- 📝 **DOCX Отчеты:** Создание красивых, читаемых документов для печати или быстрого ознакомления.
- 🎙️ **Speech-to-Text (STT):** Встроенная интеграция с Whisper для автоматической расшифровки голосовых сообщений и «кружочков».
- 🖼️ **Медиа-архив:** Опциональное скачивание фото и видео с сохранением структуры папок.
- 🔒 **Безопасность:** Программа работает локально. Ваши сессии и данные хранятся только на вашем компьютере в базе данных SQLite.

---

## 🚀 Как это работает (Use Case: Аналитика в NotebookLM)

Как упоминалось на одном yotube канале, чаты Telegram — это ценнейший источник экспертизы. Rozitta Parser позволяет превратить хаос из тысяч сообщений в структурированную базу знаний:

1. Выгрузите нужный чат в формате **Markdown** с включенной опцией **"Адаптировать для ИИ"**.
2. Загрузите полученные части (`history_part_1.md`, `_part_2`...) в **NotebookLM**.
3. Задавайте вопросы по истории чата: «Какие боли у клиентов?», «Топ-5 проблем продукта?», «Сводка мнений о конкурентах за 2024 год».

## 🛠 Установка и запуск (для разработчиков)

1. Клонируйте репозиторий:
   ```bash
   git clone [https://github.com/твой_профиль/RozittaParser.git](https://github.com/твой_профиль/RozittaParser.git)
   cd RozittaParser
Установите зависимости:

Bash

pip install -r requirements.txt
Запустите приложение:

Bash

python main.py
📦 Сборка в .EXE
Если вы хотите собрать портативную версию для Windows:

Bash

pyinstaller --noconfirm --onefile --windowed --name "Rozitta_v3.5" main.py
⚠️ Важное примечание по безопасности
При первом запуске вам потребуется ввести ваши API ID и API Hash (получить их можно на my.telegram.org).
Никогда не передавайте файлы .session и config_modern.json третьим лицам — они содержат полный доступ к вашему аккаунту!
